# Avengers Ascent: City Surge

## How to install

1. Go to the [latest release](https://github.com/mansymohammed123-hub/avengers-ascent-city-game/releases/latest).
2. Scroll down to **Assets**.
3. Click `AvengersAscentSetup.exe` to download it.
4. Open the downloaded file.
5. Click **Next**.
6. Click **Next** again (or choose install location if you want).
7. Click **Install**.
8. Click **Finish**.
9. Open the game from your Start Menu or desktop shortcut.
